## How to Use

1. **Login** - Enter your name and 4-digit PIN
2. **Check the board** - See which stands are available (green) or occupied (red)
3. **Claim your stand** - Tap an available stand to check in
4. **Log activity** - Use the Log Entry tab to record sightings or harvests
5. **Check out** - Hit the Check Out button when you leave

## Stand Board

- **Green** = Available
- **Red** = Occupied (shows hunter's name)
- **Numbers** = Stand IDs (1-20)
